Scrabble Scorekeeper

What is Scrabble Scorekeeper?

Scrabble Scorekeeper is a program I wrote for 2505-01 Visual C#.Net Programming. It can be used to store the words played and points scored in a scrabble game. It has a backend database so you can lookup historical data and statistics about any scrabble game data.
